> Jika Anda menggunakan node server.js, Anda harus mematikan dan merestart server secara manual setiap kali mengubah kode. Agar server otomatis restart sendiri saat kode disimpan, Anda bisa menggunakan modul nodemon.
<br>


>
```nodejs
"scripts": {
  "start": "node server.js",
  "dev": "nodemon server.js"
}

```
> Tambahkan script baru (misalnya bernama "dev") di dalam package.json
